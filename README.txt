Приложение является простым аналогом редактора текста.
Есть возможность редакции и форматирования текста. Сохранения в форматах .txt и .rtf.
Есть стандартные функции текстового редактора, автосохранение и изменение темы приложения.
Интерфейс интуитивно понятен из-за схожести с подобными редакторами.
